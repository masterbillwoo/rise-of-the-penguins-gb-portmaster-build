# Godot Boy

Godot Boy is a Game Boy and Game Boy Color emulator addon for Godot Engine.

For PortMaster builds, place the appropriate Linux .so files for your architecture in this directory:
- libgodot_boy.linux.aarch64.so (for ARM64 devices)
- libgodot_boy.linux.x86_64.so (for x86_64 devices)

The .so files can be obtained from the original Godot project's addons/godot_boy directory.
